### Hey, I'm Chirag Patnaik

I build browser-native tools that run entirely on your device. No servers, no API keys, no data leaving the browser.

**What I'm working on:** single-file web apps — each one HTML file with everything inline. ML inference via [Transformers.js](https://huggingface.co/docs/transformers.js), game AI from scratch, all cached after first load.
